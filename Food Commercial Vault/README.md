# Food Commercial Vault for Seedance 2.0

This package contains a compact but complete Obsidian-ready markdown vault for designing premium short-form food commercials.

## Folder overview
- `00_HUB` — navigation and usage rules
- `10_CORE` — universal food-commercial principles
- `20_PLAYBOOKS` — dish-type modules
- `30_TEMPLATES` — practical working templates
- `90_RU_APPROVAL` — Russian approval summaries

## Recommended starting files
- `00_HUB/00_Master_MOC.md`
- `10_CORE/10_15s_Commercial_Architecture.md`
- `10_CORE/11_Visual_Deliciousness_and_Appetite_Principles.md`
- `10_CORE/16_Continuity_Consistency_and_State_Tracking.md`

## Suggested workflow
1. Read the master MOC.
2. Fill the creative brief.
3. Fill the shot functions template.
4. Lock the dish identity card.
5. Add a continuity state log if the sequence contains irreversible actions.
6. Compress into a final prompt using the prompt assembly rules.
